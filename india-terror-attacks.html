Terror attacks in India: stitched historical trend and ideology share

I rebuilt this page with multiple sources. There is still no single, clean, machine-readable nationwide series that gives India terror-attack counts on exactly the same definition from 1947 to 2026. So this page now uses a stitched trend for the main count graph and keeps a separate, stricter attribution-share graph for the GTD-derived 1+ death subset.

Method

What changed

You asked for different sources to be incorporated from 1947 to 2026. I therefore switched the first graph to a stitched source approach instead of relying only on the earlier GTD-derived mirror. The tradeoff is that the main count graph is now best read as a historical trend assembled from multiple sources rather than one perfectly uniform dataset.

Main count graph

The first graph now combines Wikipedia's annual incidents table for 1970–2018 with SATP's terrorism-related incidents series for 2019–2026. I did this because it extends the visible trend much later than the earlier page. For 1947–1969 I still did not find a comparably structured annual nationwide source I trust enough to turn into a hard numeric series, so that segment remains marked as unavailable rather than invented.[1][2]

Religion / ideology share graph

The second graph remains separate and stricter. It still uses the GTD-derived India incident mirror filtered to attacks with at least one death and then classifies attributed perpetrator labels into broad ideology / religion buckets. That share graph therefore remains best interpreted for the 1979–2016 span covered by that source.[3][4]
