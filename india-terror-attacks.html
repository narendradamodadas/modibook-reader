How the main count graph is stitched now

You pointed out that the earlier plotted values did not visibly follow the SATP incidents page. That was fair. I therefore rebuilt the first graph so SATP is used directly for the full 2000–2026 range, while Wikipedia is only used for 1970–1999. The GTD-derived mirror is now used only for the separate share graph below.

Main count graph

1970–1999 comes from Wikipedia's annual incidents table. 2000–2026 comes directly from SATP's incidents series. 1947–1969 remains a clearly marked no-data zone because I have not found a nationwide annual series for those years that is comparable enough to the later sources.[1][2]

Share graph

The religion / ideology share graph is a different thing. It remains based on the stricter GTD-derived 1+ death subset, so it still covers only the years supported by that source. I keep it separate so I do not pretend the stitched count graph and the share graph come from the same dataset.[3][4]
