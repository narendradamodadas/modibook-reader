Modi Book Reader

This site hosts scanned books converted into a readable web format. Each book can include the original scanned page, OCR text, and an English translation so readers can compare the source directly.

Open Modibook Source repository

How it works

Books are converted from scanned PDFs into HTML reading pages. The scan remains visible, and the extracted text is translated for easier browsing in English.

Current collection

This library is now structured to support multiple books. More titles can be added later as separate entries without mixing all pages into one homepage.

Reading note

Because these books come from OCR and machine translation, some pages will be rough. The original page image is always available for verification.
